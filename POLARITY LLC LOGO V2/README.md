## Dimensions

- **Aspect Ratio:** 16:9
- **Format:** SVG (vector - infinitely scalable)

## How to Export to PDF

### Method 1: Print from Browser (Easiest)

1. Open `polarity-logo-4K-print.html` in Chrome or Safari
2. Press `Cmd + P` (Mac) or `Ctrl + P` (Windows)
3. Select **"Save as PDF"** as destination
4. Set paper size to **Custom: 3840 × 2160 px** or **Tabloid/A3**
5. Set margins to **None**
6. Enable **Background graphics**
7. Click **Save**

### Method 2: Export SVG to PDF (High Quality)

**Using Preview (Mac):**
1. Open any `.svg` file in Safari
2. Press `Cmd + P` → Save as PDF

**Using Adobe Illustrator:**
1. Open the SVG file
2. File → Save As → PDF

**Using Inkscape (Free):**
1. Open the SVG file
2. File → Save As → PDF

### Method 3: Command Line (Automated)

```bash
# Using Chrome headless
"/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" \
  --headless --print-to-pdf="polarity-logo-4K.pdf" \
  --print-to-pdf-no-header \
  "file://$(pwd)/polarity-logo-4K-print.html"
```

## 💾 How to Export

1. Open `export-profile-pics.html` in your browser
2. Click **"Export as PDF"** on any card
3. In print dialog:
   - Select **"Save as PDF"**
   - Set margins to **"None"**
   - Enable **"Background graphics"**
4. Save with naming format: `[brand]-profile-[size].pdf`
   - Examples: `polarity-profile-400.pdf`, `waxfeed-disc-400.pdf`

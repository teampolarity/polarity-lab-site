## 🎨 Converting to Other Formats

To convert PDFs to PNG/JPG:

### Using Preview (macOS):
1. Open PDF in Preview
2. File → Export
3. Choose PNG/JPG format
4. Set resolution (72 DPI for web, 300 DPI for print)

### Using Command Line:
```bash
# Convert to PNG at 400x400
sips -s format png polarity-icon-400x400.pdf --out polarity-icon-400x400.png

# Convert to JPG at 800x800 (2x)
sips -s format jpeg -s formatOptions 100 -z 800 800 polarity-icon-400x400.pdf --out polarity-icon-800x800.jpg
```

### Using Online Tools:
- [CloudConvert](https://cloudconvert.com/pdf-to-png)
- [Adobe Acrobat Online](https://www.adobe.com/acrobat/online/pdf-to-jpg.html)

## 🚀 Quick Upload Guide

1. **Download** the PDF you need
2. **Convert** to PNG if required (most platforms accept PNG)
3. **Upload** to your platform:
   - Twitter/X: Settings → Profile → Profile Picture
   - LinkedIn: Me Icon → View Profile → Camera Icon
   - Instagram: Profile → Edit Profile → Change Profile Photo

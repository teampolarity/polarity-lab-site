# Pain Points Logo Assets

A brutal, visceral logo featuring a needle piercing through a globe - representing global pain points.

## Logo Files

### Primary Logos
- `painpoints-logo.svg` - Full color logo (512x512) - Light backgrounds
- `painpoints-logo-dark.svg` - Dark mode version (512x512)
- `painpoints-logo-mono.svg` - Monochrome black version
- `painpoints-logo-mono-white.svg` - Monochrome white version

### Icons & Variations
- `painpoints-icon.svg` - Simplified icon (64x64)
- `painpoints-navbar.svg` - Navbar-optimized version (32x32)
- `favicon.svg` - Favicon (32x32)

### Social Media
- `og-logo.svg` - Open Graph / social sharing (1200x630)

## Brand Colors

- **Pain Red**: `#E63329`
- **Black**: `#0A0A0A`
- **White**: `#FAFAFA`

## Usage

### Navbar (inline React component)
The navbar uses an inline SVG component. See `src/components/layout/Navbar.tsx` for the `PainPointsLogo` component.

### Favicon
Copy `favicon.svg` to `public/favicon.svg` for browser tab icon.

### Social Sharing
Use `og-logo.svg` as the `og:image` meta tag for social media previews.

## Concept

The logo represents:
- **Globe**: Global reach, worldwide problems
- **Needle**: Pain, precision, pointing to specific issues
- **Pierce marks**: Impact, the visceral nature of problems
- **Grip lines**: Control, taking hold of issues

The diagonal needle creates dynamic tension and forward momentum.
